## Installation
```
go get github.com/gofiber/fiber/v2
```
```
go get github.com/gofiber/fiber/v2/middleware/cors
```
```
go get github.com/joho/godotenv
```

## Install Air dependency, this is equivalent to nodemon in node.js.
```
Go to this page, and read the author's ReadMe

https://github.com/air-verse/air
```

## In the config.go file, change password to your password in your postgresql database.
```
  connStr := "user=postgres dbname=postgres password=" + password +  /*your password*/  " sslmode=disable"  
```
